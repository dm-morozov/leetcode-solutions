# LeetCode Solutions

В этом репозитории я решаю задачи с LeetCode.

Основная цель — прокачка алгоритмического мышления, подготовка к собеседованиям и поддержание навыков программирования.

## Языки

Основной язык:

- Python (приоритет, планирую развиваться как Python-разработчик)

Дополнительно:

- TypeScript (использую для практики типизированного подхода)
- JavaScript (реже, для поддержки и сравнения)

TypeScript для меня приоритетнее JavaScript, так как это типизированный язык, и мне комфортнее работать с типами.

В Python также использую аннотации типов (`typing`), чтобы писать более чистый и понятный код.

## Как я решаю задачи

- Сначала стараюсь решить задачу самостоятельно
- Не копирую чужие решения
- Важно понять логику, а не просто получить ответ
- Иногда добавляю комментарии на русском и английском

## Структура репозитория

Задачи разбиты по темам:

- hash_table
- math
- arrays
- strings
- two_pointers
- linked_list
- binary_search
- trees
- dynamic_programming

## Формат решений

Каждый файл обычно содержит:

- номер задачи
- название
- сложность
- темы
- оценку сложности (Time / Space)
- краткую идею решения

Пример:

```python
# 001. Two Sum
# Difficulty: Easy
# Topics: Array, Hash Table
# Time: O(n)
# Space: O(n)
```
